# Enquirer

Enquirer is the library I'm using to prompting the user for information about their new module.

## Available Prompts

This list shows the various types of prompts that are available by default:

- [Auto-Complete][0]
- [Basic-Auth][1]
- [Confirm][2]
- [Form][3]
- [Input][4]
- [Invisible][5]
- [List][6]
- [Multi-Select][7]
- [Numeral][8]
- [Password][9]
- [Quiz][15]
- [Survey][10]
- [Scale][11]
- [Select][12]
- [Snippet][14]
- [Sort][13]
- [Toggle][16]
